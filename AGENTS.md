# ContextForge — Agent Orchestrator

This file is an **index only**. It lists project skills and where to find them.
Do not put workflow instructions here — those belong in each skill's `SKILL.md`.

## How to use

1. Read this index to find a relevant skill.
2. Open the skill file at the path listed below.
3. Follow the instructions in that `SKILL.md`.

## Register a new skill

1. Copy `skills/_template/` to `skills/<skill-name>/`.
2. Edit `skills/<skill-name>/SKILL.md` (frontmatter + instructions).
3. Add a row to the table above.
4. See `skills/README.md` for conventions.
